# itp access-control permissions me

Retrieve a list of your permissions on a specified iTwin.

## Options

- **`--itwin-id`**  
  The ID of the iTwin for which the role is being created.  
  **Type:** `string` **Required:** Yes

## Examples

```bash
itp access-control permissions me
```
